# GeoMonitor 프론트엔드

> 지반 계측 센서 실시간 모니터링 시스템 - 프론트엔드

## 📋 프로젝트 개요

GeoMonitor는 지반 계측 센서 데이터를 실시간으로 수집·저장·시각화하는 모니터링 시스템입니다.
수위계, 지중경사계 등 다양한 센서의 측정값을 1시간 단위로 자동 수집하며, 임계값 초과 시 알람을 발생시킵니다.

## 🔗 배포 URL

- **프론트엔드**: https://yuhyun-sensor-monitoring-front.vercel.app
- **백엔드 API**: https://yuhyun-sensor-monitoring-back.onrender.com
- **Swagger UI**: https://yuhyun-sensor-monitoring-back.onrender.com/api-docs

## 🛠 기술 스택

- **Framework**: Next.js 16
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **차트**: Recharts
- **PDF 출력**: jsPDF + jspdf-autotable
- **엑셀 출력**: ExcelJS
- **배포**: Vercel

## 📱 주요 기능

### 대시보드
- 전체 센서 현황 KPI 카드 (전체/정상/주의/위험/오프라인)
- 최근 알람 목록
- 1분마다 자동 갱신
- 새로고침 버튼 (갱신 중 상태 표시)
- **데이터 수신 지연 경고**: 2시간 이상 데이터 미수신 센서 발생 시 경고 배너 자동 표시, 수신 재개 시 자동 해제

### 센서 관리
- 모니터링 / 센서 정의 / 계산식 관리 / 재수집 4개 탭
- 상태 필터 (전체/정상/주의/위험/오프라인)
- **데이터 수신 지연 경고**: 2시간 이상 미수신 센서 경고 표시
- 센서 상세 페이지, QR 코드
- 계산식 관리 탭: 관리자만 추가/수정/삭제 가능
- **재수집 탭**: 에이전트 온라인/오프라인 상태 확인, 재수집 요청 등록 및 이력 조회
- 센서 편집 모달: 계수값(A, B, C, D, G 등) 저장 후 재편집 시 정상 불러오기

### 센서 상세 페이지
- 시간별/일별 트렌드 차트 (X축 날짜 오름차순 정렬)
- 1차 상한/하한 관리기준선 동시 표시 (Y축 자동 포함)
- 기준선 라벨에 기준값 함께 표시 (예: `1차 하한기준 (-4.00)`)
- Y축 레이블 소수 2자리 반올림
- 날짜 범위 선택 (오늘/7일/30일/90일 프리셋)
- 측정 데이터 테이블 (경과일 자정 기준 계산)
- 초기측정값 고정 표시 (날짜 범위 무관, 전체 기간 최초 수신값)
- 보고서 출력 모달 (Excel 다운로드 / PDF 다운로드)
- 계측계획 평면도 카드 (센서별/현장별 PNG/JPG/PDF 업로드, base64 DB 저장)
- 계산식 상수값 표시 (formula_params에서 읽어 센서 정보 카드에 표시)
- 80053 전용: 기간 내 최신값/초기측정값 Linear/Poly 전환
- 80053 전용: Linear initRaw 날짜 범위 무관 전체 기간 첫 번째 raw값으로 고정

### 80053 수위계 전용 기능
- **depth_label 선택**: 1번 수위계 / 2번 수위계 / 3번 수위계 버튼
- **계산식 선택**: Linear(메인) / Polynomial 토글
- **depth별 보정값 입력**: depth별 독립적으로 보정값 입력 및 DB 저장 (correction_params)
  - 보정값 적용 시 차트/측정데이터/초기측정값/최솟값/최댓값 모두 반영
  - 새로고침 후에도 보정값 유지 (DB 저장)
- **1차 상한/하한 자동 계산**: 초기측정값 ± 4m로 표시
  - 센서 정보 카드: `4.00 m (초기값 0.00 + 4m)` 형식으로 표시
  - 차트 기준선에도 동일하게 반영
  - 보정값 변경 시 상한/하한도 연동하여 자동 갱신
- psi→m 변환계수: 0.70307
- Linear: `P(psi) = G × (초기값 - 현재값)` × 0.70307
  - depth 1번: G=0.012044
  - depth 2,3번: G=0.013450
- Polynomial: `P(psi) = A×R² + B×R + C` × 0.70307
  - depth 1번: A=7.080E-08, B=-0.012296, C=106.0458
  - depth 2,3번: A=1.429E-07, B=-0.015320, C=118.4773

### QR 페이지 (`/qr/[id]`)
- 로그인 없이 센서 현재 상태 조회
- 80053 센서: depth 1/2/3번 각각 Linear(메인) / Polynomial 최신값 표시
- 보정값 적용된 최신값 표시
- 일반 센서: 기존 현재 측정값 표시

### 엑셀 출력 (Water Level Meter Report)
- 헤더 정보 (현장명/계측기 No./설치현황/초기측정일/관리자+권한/설치위치)
- 초기측정일: 전체 기간 최초 수신값 기준 (globalInitReading)
- 브라우저 차트 이미지 삽입 (SVG → PNG 변환, 6~15행)
- 범례: 센서명 / 1차 하한기준(빨간) / 1차 상한기준(주황) 분리 표시
- 데이터 표 (측정일/경과일/지하수위/전측정치대비/초기치대비/비고)
- **날짜 하루 선택 시**: 시간별 데이터(measurements) 전체 출력
- **날짜 범위 선택 시**: 일별 데이터(dailyReadings) 출력
- 초기치 행 강조 (노란 배경 + 빨간 텍스트)
- A4 인쇄 설정 자동 적용

### PDF 출력 (Water Level Meter Report)
- 헤더 정보 (관리자 이름+권한 포함)
- 초기측정일: 전체 기간 최초 수신값 기준 (globalInitReading)
- 차트: 1차 상한/하한 기준선 포함 (80053은 초기값 ± 4m 기준)
- 범례: 센서명 / 1차 하한기준(빨간) / 1차 상한기준(주황)
- **날짜 하루 선택 시**: 시간별 데이터 전체 출력
- **날짜 범위 선택 시**: 일별 데이터 출력
- NanumGothic 한글 폰트 적용
